# Agrinho2026
 O usuário que acessar este site, deve ler com atenção as informações apresentadas, absorver o conteúdo, e depois deixar sua opinião através dos botões de like e dislike.
 O meu principal objetivo é, através de conceitos resumidos, porém explicados, lhe informar e fazer com que você aprenda o que é, os objetivos, e como a agricultura de precisão reflete sobre o agronegócio brasileiro. 
 O site possui acessibilidade ao possuir um texto em língua inglesa, possibilitando que outros indivíduos possam lê-lo. Além disso, quis interagir com o usuário através dos botões, pois acredito ser uma maneira simples e divertida de se expressar.
